# 📬 Gen7 - A URL Shortener Service

This is a URL Shortening Implementation.

---

## 🚀 Features

- Serve as a basic shortener for extremely long URLs


---

## 📦 Installation

```bash
git clone https://github.com/dotandev/gen7.git
cd gen7
npm install

```


### ▶️ Run the app

```bash
npm run begin
```
